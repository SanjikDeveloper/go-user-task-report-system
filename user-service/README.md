# Pet-project user-service

-------------------

## Описание

Сервис управления пользователями с регистрацией, авторизацией через JWT-токены.

-------------------

## Стек

Реализован на Go с использованием Gin, PostgreSQL и JWT.

-------------------

## Запуск проекта

Для запуска проекта требуется написать команду в терминал
```go run cmd/app/main.go```
