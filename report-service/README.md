# Pet-project report-service

-------------------

## Описание

Сервис генерации PDF-отчетов.

-------------------

## Стек

Реализован на Go с использованием Gin, PostgreSQL и Redis

-------------------

## Запуск проекта

Для запуска проекта требуется написать команду в терминал
```go run cmd/app/main.go```
