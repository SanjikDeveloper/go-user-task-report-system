# Pet-project - tasks-service

-------------------

## Описание

Сервис управления задачами. Поддерживает CRUD-операци

-------------------

## Стек

Реализован на Go с использованием Gin и PostgreSQL. 

-------------------

## Запуск проекта

Для запуска проекта требуется написать команду в терминал
```go run cmd/app/main.go```
